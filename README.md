# 🎬 YouTube Ad Revenue Predictor

## 📌 Project Overview
Predicts YouTube Ad Revenue based on 
video metrics using Machine Learning.

## 📊 Model Performance
- Algorithm : Random Forest Regressor
- R² Score  : 94.36%
- MAE       : $5.25
- Dataset   : 122,400 records

## 🔧 Tech Stack
- Python
- Scikit-learn
- Pandas & Numpy
- Streamlit

## 🚀 How to Run
pip install -r requirements.txt
streamlit run app.py

## 💡 Key Insight
Watch Time is the #1 driver of Ad Revenue
with 93.7% Feature Importance!
